## 🌟 Overview
A meticulously crafted collection of React components tailored specifically for the Persian web ecosystem. From Jalali date pickers to Toman currency formatters, this library saves you hours of dealing with RTL quirks and localization issues.

## ✨ Key Features
- 🗓️ **Jalali Calendar Support**: Seamless Persian date handling.
- 💰 **Currency Formatting**: Built-in Toman/Rial formatters with standard grouping.
- 📐 **RTL Optimized**: Pixel-perfect right-to-left layout compliance.

## 🚀 Getting Started

### Prerequisites
Make sure you have `React` installed on your local machine.

### Installation
```bash
git clone https://github.com/HafezMosleh/persian-components-react.git
cd persian-components-react
npm install persian-components-react
```

### Usage
```bash
import { PriceFormatter } from 'persian-components-react';

<PriceFormatter price={150000} currency="تومان" />
```
