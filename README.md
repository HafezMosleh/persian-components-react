# persian-components-react

A set of high-quality React components for Persian/RTL (Datepicker, Price Formatter).

## Features
- 🚀 Fast and optimized
- 🔒 Secure by default
- 🛠 Easy to configure

## Installation
```bash
git clone https://github.com/HafezMosleh/persian-components-react.git
cd persian-components-react
```
